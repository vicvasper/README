Proyectos Destacados y Open Source

Kinemotion Mocap C++ / NNE / LIVE LINK

Plugin de mocap monocular en tiempo real para UE5.3+. Ejecuta inferencia de pose IA vía NNE (DirectML GPU), decodifica 133 keypoints en un esqueleto de 19 huesos y transmite a través de una fuente Live Link personalizada con auto-calibración, filtrado anti-jitter y soporte multijugador. GitHub

Más Proyectos

Optilogger Plugin C++ / SLATE / PROFILING

Plugin de profiling para editor (+1.100 líneas de análisis, 865 líneas de Slate UI) que escanea meshes, texturas, materiales, animaciones, audio, iluminación y post-proceso. Estimación de memoria, conteo de instrucciones de shader, culling de frustum y exportación JSON. ★6 estrellas. GitHub

SmartFolders Plugin PYTHON / C++ / EDITOR

Organizador de assets por reglas para el Content Browser. Dos modos (basado en tipo + agrupación inteligente por nombre), presets recomendados por Epic, undo completo y operaciones en lote. Reduce la organización manual en ~70%. ★4 estrellas. GitHub

Paint System C++ / RENDER TARGET / MATERIALS

Pintado en tiempo real por color de vértice para Static y Skeletal Meshes. RGBA por canal, fundido configurable, actualizaciones por lotes en GPU y API de Blueprint. ★5 estrellas. GitHub

Procedural Animated System C++ / SPLINE / PCG

Generador procedural con conciencia del entorno. Usa trazados de línea sobre el terreno para generar puentes de spline, escaleras en zigzag o rampas escalonadas según las normales de superficie. Listo para animación con spawn en tiempo de ejecución. GitHub

Time Control Mechanic C++ / PHYSICS / GAMEPLAY

Manipulación temporal por objeto: avance rápido, rebobinado y congelación. Captura y restaura el estado físico completo por actor para un control temporal granular. GitHub
