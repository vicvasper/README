# Kinemotion Plugin (UE5.5+)

TL;DR
Kinemotion provides a simple pipeline to capture body poses (MediaPipe) and map them to Unreal skeletal meshes in real-time. Includes demo project and Blueprint bindings for rapid prototyping.

Features
- Real-time MediaPipe integration
- Skeletal retargeting and smoothing
- Blueprint examples and demo scene
- Multiplayer-friendly replication hooks (example)

Quickstart
- Download or clone the plugin into Plugins/, open the demo project, and press Play with webcam enabled.

Role & impact
- Built the pipeline for real-time mapping and demo integration; useful for prototyping mocap-driven gameplay.
