# SmartFolders Plugin (UE5.6)

TL;DR
SmartFolders reduces Content Browser clutter by automatically organizing and grouping assets via rules and heuristics. Designed to speed up large project workflows and keep content tidy.

Features
- Rule-based auto-grouping of assets
- Custom identifiers and tag-based organization
- Batch rename & move operations with undo support
- Lightweight UI integrated into the editor

Quickstart
- Install in Plugins, enable, and open the SmartFolders panel.
- Create a rule (e.g., textures -> /Textures/, skeletal meshes -> /Characters/).

Role & impact
- Implemented the core organization engine and user-facing rules UI.
